# MlSearch
## Proyecto desarrollado en angular
Angular CLI version 10.1.1.

### Setup
Clonar repositorio e iniciar el servidor con
#### npm run start

### Para correr pruebas
npm run test

## Si tienes istalado Docker
1. clonar repositorio
2. inciar docker compose:
#### docker-compose up --build
